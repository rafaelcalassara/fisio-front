# Fisio Front

Protótipo estático de frontend para uma aplicação de operação clínica de fisioterapia/pilates.

## Requisitos mapeados até agora

### Explícitos
- Existe um projeto de sistema de fisioterapia para o madrinho do Rafael.
- A aplicação precisa ser publicável sob `rafaelcalassara.com.br`.
- O subdomínio solicitado para esta versão é `fisio.rafaelcalassara.com.br`.
- A preferência operacional é por stack simples, barata e fácil de publicar.

### Inferidos para o v1
- Agenda diária/semanal.
- Cadastro simples de pacientes.
- Evolução/prontuário por sessão.
- Planos terapêuticos/exercícios.
- Financeiro mínimo para acompanhar cobrança e receita.

## Loop de desenvolvimento sugerido
1. Mapear fluxo real da clínica.
2. Desenhar telas simples e validar com o usuário final.
3. Entregar um slice operacional pequeno.
4. Medir uso real e gargalos.
5. Refinar antes de ampliar escopo.
6. Escalar só depois para automações e relatórios.

## Arquivos
- `index.html`
- `styles.css`
- `script.js`

## Rodar localmente
Abra `index.html` ou sirva a pasta com um servidor estático.
